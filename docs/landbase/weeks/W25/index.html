上週焦點頭條

Zitro FANTASY 機櫃一口氣推出五款獨家新作

Zitro 在「Zitro Experience Peru」一次端出主打 FANTASY 機櫃的五款獨家新作：Brave Dragon、Lion Falls、Gold Space、Fantasy Mine、Rock n' Raccoons；CONCEPT 機櫃同步預覽 Penguin Party、Treasure Shore、Ancient Link 等多款遊戲庫。FANTASY 主打華麗視覺與沉浸式多媒體，是 Zitro 拓展拉丁美洲市場的核心機型，本次大規模上新預示其後續在祕魯與全拉美的鋪機動能。

三大主廠焦點

Aristocrat

Aristocrat 推出 Class II 新機《Mr. & Ms. Money Bags》

Aristocrat Gaming 發表全新 Class II 機台《Mr. & Ms. Money Bags》，採用 Marquis 五軸機櫃搭配實體轉軸與數位頂部轉盤，並串接 Frenzy Jackpots 連線彩金系統。遊戲內含四種 Bonus：基本盤的 VGT 經典 Red Screen 免費遊戲、可變型（metamorphic）的 Super Red Screen 免費遊戲乘數、可變型轉盤乘數，以及標準轉盤獎勵；兩種可變型 Bonus 可單獨或同時觸發，連動時依序先跑乘數再進轉盤。鎖定部落（tribal）與區域型 Class II 市場，延續 Aristocrat 在該品類的布局。
